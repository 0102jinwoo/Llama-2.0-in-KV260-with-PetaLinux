This repository explains the entire process of implementing the Llama 2.0 (110M) model using HLS and running it on the KV260 board using PetaLinux.

### Prerequisites
To follow along with this project, you will need the following:

**Software & Environments:**
* Llama 2.0 (110M) model weights and tokenizer
* Vitis 2025.2
* Vivado 2025.2
* KV260 BSP (v2025.1)
* PetaLinux 2025.2
* Ubuntu 24.04 (Host environment)
* Ubuntu 22.04 (Container environment for PetaLinux)
* Win32 Disk Imager
* Tera Term

**Hardware:**
* Kria KV260 Vision AI Starter Kit
* MicroSD card
* USB-UART Console Cable
* 12V AC Power Adapter

I hope this guide helps you in successfully running FPGA boards utilizing PetaLinux!
